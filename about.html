FoneLoan

FoneLoan is a new innovation in the field of Nepal’s banking environment targeted for salary holders of any institution. This application can be used as a plugin/standalone application which allows client’s to apply for short term loan. This application runs decision analytics in the background to proceed further. Ultimately the end user can receive short-term loan instantly which will be deducted automatically from the end users account on the agreed loan expiry data. This application has been adopted by top 3 'A' class banks (Kumari, Laxmi and Nabil bank) in Nepal. Tech Stack: Java, Spring Boot, Angular 7, MySQL, Jenkins.
